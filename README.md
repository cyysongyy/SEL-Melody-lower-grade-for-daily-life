🌈 和平小島魔法石 (Peace Island SEL)
這是一個專為低年級學童設計的 社會情緒學習 (SEL) 互動素材包。透過 32+1 張可愛動物去背圖，引導孩子練習行為規範與情緒調節。

📦 素材內容
五大時段行為 (25張)：涵蓋早安出發、午後靜心、活動降溫、聯絡簿時間與快樂回家。

情緒調節工具：冷靜氣球、生氣面具、和平小島背景。

遊戲介面組件：開始/成功按鈕、回首頁圖示、承諾勾勾手、感謝能量球。

🚀 如何使用
下載素材：將 images/ 資料夾內的 PNG 檔匯入 PPT。

建立頁面：利用背景圖結合行為素材，設計互動情境。

教學應用：引導學生點擊按鈕，模擬情緒降溫與行為選擇。
